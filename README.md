🚦 Intelligent Traffic-Detecting Robot

> AI-powered robot that detects traffic lights and road signals using OpenCV and computer vision.

## Tech Stack
- Python
- OpenCV
- ROS2
- Raspberry Pi 4
- Camera Module

## Features
- Real-time traffic light detection
- Color recognition (Red/Yellow/Green)
- Autonomous stop/go behavior
- ROS2 integration

## Project Structure
- src/    — Source code
- docs/   — Documentation
- media/  — Photos and videos
